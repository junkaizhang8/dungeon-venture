# Dungeon Venture

## Install
1. Have [CMake](https://cmake.org/download/) installed
2. Clone the repository
    ```
    git clone https://github.com/junkaizhang8/dungeon-venture.git
    ```
3. Initialize and fetch submodules
    ```
    git submodule update --init --recursive
    ```
4. Build the project
    ```
    ./build.sh
    ```
5. Run the editor
    ```
    ./run_editor.sh
    ```
